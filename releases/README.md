# Releases

This directory stores formal source-and-results release snapshots for the
restored AbTweak experiment environment.

Each release directory should match:

- the checked-in [VERSION](/Users/stevenwoods/mmath-renovation/VERSION)
- the matching [CHANGELOG.md](/Users/stevenwoods/mmath-renovation/CHANGELOG.md)
- an annotated git tag such as `v0.10.0-beta.1`

The snapshots are generated by
[scripts/create-release-snapshot.sh](/Users/stevenwoods/mmath-renovation/scripts/create-release-snapshot.sh).
