# Recovery Artifacts

This directory holds retained local working artifacts that are worth preserving
for project continuity even when they were not promoted directly into the main
document set in their original form.

Examples include:

- local uncommitted patches preserved before cleanup
- one-off migration notes
- recovery-time evidence about how the working tree was stabilized

Policy:

- prefer promoting polished content into its main long-term home
- preserve non-polished but potentially useful local work here rather than
  letting it disappear during cleanup
- treat this directory as part of the continuity surface of the repo
