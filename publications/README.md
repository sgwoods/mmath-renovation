# Publications

This directory holds checked-in reference copies of the core historical AbTweak publications used throughout this repository.

## Naming Scheme

Files use:

`YYYY-source-short-title.ext`

Where:

- `YYYY` is the publication year when it is known
- `source` is the issuing venue or archive
- `short-title` is a compact slug for the document

If the original page is archival or undated, the filename uses the source/archive name without forcing a year.
